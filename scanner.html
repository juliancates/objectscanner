How it works
44 fiducial dots — The mat border has filled circles at precisely known physical positions (0.5in spacing, 0.35in inset). Even if 15+ dots are obscured, the rest are sufficient.
Blob detection — OpenCV finds all dark circular blobs, filtered by area, circularity (>0.6), and solidity. Candidates are matched to the known dot grid using estimated scale.
RANSAC homography — cv.findHomography with RANSAC fits the perspective transform across all matched points, automatically rejecting any mismatched dots.
4× warp + CLAHE — Image warped to 3264×4224px (384dpi) via Lanczos4. CLAHE normalises local contrast before thresholding. Grid lines suppressed via morphological open on inverted image.
Solid SVG output — Contours traced with hierarchy (holes cut through via evenodd fill), Catmull-Rom bezier smoothing, RDP simplification. Exact physical dimensions: width="8.5in" height="11in".
